# TomoGroupFrame

TomoGroupFrame is a World of Warcraft addon designed to enhance group and raid frame management. It provides customizable unit frames, advanced tracking features, and a user-friendly configuration interface.

## Features

- Customizable party and raid frames
- Bar textures and icon support
- Dispel and HoT tracking modules
- Profile management for different setups
- Localization support (English, French)
- Easy-to-use configuration UI

## Folder Structure

- `Assets/`: Contains fonts and textures for bars and icons
- `Config/`: Configuration UI, widgets, and panels for party, raid, and profiles
- `Core/`: Database, initialization, profiles, and utility functions
- `Locales/`: Localization files for supported languages
- `Modules/`: Main addon modules (BarTextures, DispelTracker, HoTTracker, PartyFrames, RaidFrames, UnitFrame)

## Installation

1. Download or clone the addon repository.
2. Copy the `TomoGroupFrame` folder into your WoW AddOns directory:
   - `World of Warcraft\_retail_\Interface\AddOns\`
3. Restart World of Warcraft or reload your UI.

## Usage

- Access the addon configuration via `/tomogroup` or through the game interface options.
- Customize your group and raid frames, select textures, and manage profiles.
- Modules can be enabled or disabled as needed.

## Localization

- English (`enUS`) and French (`frFR`) supported.
- Contributions for other languages are welcome.
